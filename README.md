# next-prototyping-tamplate

This is a next.js template for rapid prototyping.

## Enviroments

- Next.js
- Typescript
- Emotion
- tailwindcss
  - twin.macro
- eslint
- prettier
- lefthook

## Getting Started

Create project:

```bash
yarn create next-app --example "https://github.com/Cilvia333/next-prototyping-tamplate" <your-project-name>
```

Run dev server:

```bash
yarn dev
```
